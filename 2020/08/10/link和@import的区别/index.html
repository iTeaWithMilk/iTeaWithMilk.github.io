页面导入样式时，link 和@import 的区别

  • link 属于 HTML 标签，除了加载 CSS 以外，还能用于定义 RSS，定义 rel 链接等属性，而@import 是 CSS 提供的，只能用于加载 CSS
  • 当解析到 link 时，页面会同步加载 link 引入的 CSS，而@import 所引用的 CSS 要等到页面加载完才会被加载
  • @import 需要 IE5 以上才能使用，link 是 HTML 标签，无兼容问题
  • link 可以使用 js 动态引入，而@import 不行
